⌨ mycora :

The command prompt — every command mycora recognizes, in one place
v0.15.0  ·  MIT OR Apache-2.0
crates.io/crates/mycora
github.com/TSODev/mycora
⌨ The Prompt Itself
:Open the command prompt from Normal mode
(type)Edit the command by hand
↑ ↓Move a cursor through the popup below, filling the prompt with the highlighted entry's syntax
EnterRun the command — or, right after arrowing to a pick, just hide the popup and leave the text to finish typing; a second Enter then runs it
EscCancel without running anything
🔗 Reindex & Broken Links
:reindexRebuild the search index for every mounted vault
:brokenlinksList every broken wikilink across every mounted vault, with a best-guess fix; jump to one to fix it by hand
🔀 Import · Export · Language · Quit
:export <path>Flatten the selected note's subtree to Markdown or PDF (by extension)
:import <path>Create a child note from an external Markdown file
:lang <en|fr|es|de>Switch the interface language — persists to config.toml
:q, :quitQuit — same as q q in Normal mode
🏷 Tags
:tags <t1,t2,...>Notes matching any of the given tags, across every mounted vault
:tags listEvery distinct tag, alphabetically, note counts summed — Enter filters by one
:tags limit <vault>Restrict :tags/:tags list to one mounted vault
:tags unlimitLift a :tags limit, back to every vault
:tag add <tag>Add a tag to the selected note
:tag del <tag>Remove a tag from the selected note
🗂 Vaults & Layout
:panes resetReset the split layout to the default 40/40/20
:config unmount <show|hide>Show/hide unmounted-vault placeholder rows in the tree
:config archive <show|hide>Show/hide archived-vault placeholder rows in the tree
💻 Headless / CLI Equivalents
mycora reindex [--watch]Same as :reindex, outside the TUI; --watch keeps running
mycora repair [--apply]
[--create-stubs] [--vault <n>]		Batch report/fix for broken wikilinks — :brokenlinks' headless, all-at-once sibling
mycora export <title> <out>Same as :export, by exact title instead of selection
mycora vault ...add · mount · unmount · archive · unarchive ·
import · sync-filenames
💡 Tips
Unknown commandReports an error in the status bar — never silently a no-op
Popup pick shortcutArrowing to :tags <tag1,...> fills in tags — placeholders are stripped, not inserted
Config file~/.config/mycora/config.toml (Linux/macOS),
%APPDATA%\mycora\config.toml (Windows)
IndexDisposable SQLite at ~/.local/share/mycora/ — safe to delete, rebuilt on demand
PDF export fontEmbedded DejaVu Sans/Mono — Latin/Greek/Cyrillic render correctly, CJK/emoji don't yet
