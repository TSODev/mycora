# Mycora

A tree-native, mycelium-linked note-taking TUI, written in Rust.

Mycora treats two note-taking philosophies as complementary instead of
exclusive: **every note has exactly one place in a tree** (for orientation),
and **any note can also link to any other note in the forest** (for
associative reach), the way a mycelial network links the root systems of
otherwise separate trees underground.

This vault *is* Mycora's own documentation, written as a Mycora vault — a
demonstration of the tool by the tool. Use `/` to search it, `b` to see
what links where, and wikilinks throughout to jump between related notes
instead of only moving up and down the tree.
