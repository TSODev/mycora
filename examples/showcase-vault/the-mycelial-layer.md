# The mycelial layer

The name "Mycora" (from *mycorrhiza*) points at the part of
the design that's actually differentiating: not the tree — everyone has
trees — but the network that links separate trees together, the way a
mycelial network lets physically distinct trees exchange resources
underground.

In the data model, each note tree is a **trunk** — a self-contained
hierarchy you collapse, expand, move, and reorganize like a mind map.
Cross-references between notes, wherever they live in whichever tree, are
the **mycelial links** — the hidden network that connects distinct
hierarchies without flattening them into one.

This is fully implemented now: see [[Cross-links and backlinks]] for the
wikilink syntax, backlinks panel, and cross-vault resolution that realize
this half of the philosophy.
