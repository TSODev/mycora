# Rowdy

A fast, modern TUI database management tool, written in Rust.

Rowdy connects to PostgreSQL, SQLite, MySQL/MariaDB, libsql/Turso, and Redis
out of the box (MongoDB and DuckDB behind opt-in feature flags), and lets
you browse tables, edit rows, run SQL, navigate foreign keys, and export
data — all from the terminal, without a mouse.

This vault documents Rowdy as a Mycora showcase — real documentation, not
filler content, turned into interlinked notes the way Mycora itself is
documented in its own showcase-vault.
