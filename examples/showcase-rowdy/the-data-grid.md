# The data grid

Infinite-scroll pagination (200 rows per page, with a parallel
`COUNT(*)`), cumulative type-aware column filters, sortable columns
(`s` cycles ascending/descending/none), full-text search (`Ctrl+F`,
`n`/`N` to step through matches), resizable columns (`-`/`=`, `Space`
to collapse), a preview panel for untruncated cell values, and a
`[T]`/`[V]` badge distinguishing tables from views (views are
automatically read-only). The cursor survives a filter, sort, or save
— see [[The data grid remembers your row after a filter or sort]].
