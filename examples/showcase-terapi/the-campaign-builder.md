# The campaign builder

`terapi build [campaign.toml]` is a second, independent TUI for
constructing a campaign visually: a numbered pipeline with badges per
step, a "brick" catalog to add new steps, a step editor, a live
run-step preview, JSON-path autocomplete when configuring an Extract
step, loading requests straight from an existing collection, a
variables panel, a checker running 10+ static-validation rules before
you save, and a syntax-highlighted TOML preview of the exact file it
will write.
