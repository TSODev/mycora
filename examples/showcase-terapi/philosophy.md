# Philosophy

The README's own "Why terapi?" table names exactly what it's up
against: Postman/Insomnia (Electron, a cloud account required, heavy),
ATAC (a good REST TUI, but no GraphQL, no scripting), hurl (excellent
for scripting, no interactive TUI), HTTPie (terminal, but not a TUI).
Terapi's bet is that these don't have to be separate tools: GraphQL
support, pipeline automation, and an interactive TUI can all live in
one single binary.
