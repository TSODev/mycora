# The keymap cheat sheet reuses jsoned's own accent colors

The printable keyboard reference (HTML/PDF) is color-coded using
jsoned's own in-app accent colors — cyan for navigation, amber for
editing, magenta for plugins/saving, violet for diff mode — rather than
an arbitrary palette invented just for the cheat sheet, and matches the
same companion-card format terapi and rowdy use for their own reference
cards.
