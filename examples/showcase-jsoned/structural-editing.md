# Structural editing

Rename a key, add/delete/duplicate/move a node, or wrap any node in a
new Array or Object with a single keypress — the full set of
tree-shape edits `JNode`'s own structure is built to support, on top of
the plain scalar edits described in [[Editing scalars with type conversion]].
