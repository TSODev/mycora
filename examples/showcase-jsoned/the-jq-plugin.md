# The jq plugin

`|` opens a Plugins menu; the one plugin shipped today is `jq`, bundled
so no external `jq` binary is required — see [[The plugin system is narrow and compiled-in, on purpose]]. More plugins (code generation,
web import-and-prune) are named as open roadmap items, not yet built —
see [[Roadmap]].
