# Rustfmt

`cargo fmt` reformate le code selon un style canonique unique
(configurable via `rustfmt.toml`, absent par défaut) — la communauté
Rust s'est largement accordée sur un seul style, ce qui évite les
débats de formatage habituels dans d'autres écosystèmes. S'exécute
généralement en pré-commit ou en CI, aux côtés de
[[Clippy, le linter]].
